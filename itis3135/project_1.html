Project Proposal

Project Name: Grocery Store Items

The purpose of this project is to build a website for a local grocery store. In a era of digital marketing, an online presence of a grocery store will help to reach its potential customers.

  • To expand the business
  • To reach potential customers
  • The website will feature all the items in the store

Client Information

The website will be built for a local grocery store.

  • Name of the store is Grocery4U
  • Groceryforyou.com will be their domain
  • contactus@groceryforus.com

Site Map

Pages Design

I intend to make this website very minimal yet informative. The website will have more than 10 pages with different pages. It will include contact us page, home page, menu bar with more sub links. Since it is a page for a grocery store, items will be divided according to departments such as fresh food, snacks, drinks, pet food, dairy and so on. It will also feature a form where customers can sign up.

Dynamic Functionality on the Website

The website will be built using HTML, CSS, and JavaScript. It will feature images, forms, and a brand page. I will be using styles from lowesfood.com since their page has a good element of CRAP.
